# ML Assignment 2 Student Package

This folder contains everything needed for the assignment.

Files:

| File | Purpose |
|---|---|
| `ML_Assignment_2.md` | Assignment instructions. |
| `DATA_DICTIONARY.md` | Explanation of every dataset column. |
| `agent_tool_tasks.csv` | Dataset for the assignment. |

The dataset is a simplified CSV extracted from the Berkeley Function Calling Leaderboard JSON files in the Gorilla repository:

<https://github.com/ShishirPatil/gorilla/tree/main/berkeley-function-call-leaderboard/bfcl_eval/data>

You do not need the original JSON files to complete the assignment.
